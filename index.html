What it does

Everything you need.
One portable app.

NTFSecur is a single-file Windows application (.exe) built with Python + tkinter. It combines 13 system management modules in one dark Aero Glass UI — from NTFS partition security to BitLocker, process monitoring, USB tracking, backup, and more.

Getting started

Installation

Three ways to run NTFSecur — choose what works best for you.

▶ Run EXE
🐍 From Source
⚙ Build
# Option A — Portable EXE (recommended)

1. Download NTFSecur.exe from dist\
2. Right-click → Run as Administrator
3. Done — no installation required
# Option B — Run from source

pip install pillow pywin32   # optional

# Run as Administrator:
python NTFSecur.py
# Option C — Build to EXE

build.bat                    # standard build
build.bat --no-upx          # skip UPX compression
build.bat --debug           # verbose PyInstaller
build.bat --clean           # clean dist\ first

# Output:
dist\NTFSecur.exe
